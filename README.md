# Skills

Personal agent skills shared as an installable catalog.

Some skills are general-purpose, and some are agent-specific. Open each skill folder's README for install and usage details.

## Skills

| Skill | Summary |
| --- | --- |
| [`move-thread-codex`](skills/move-thread-codex/) | Move a local Codex thread to a different project/sidebar bucket by updating its stored cwd in SQLite and rollout JSONL with backups and verification. |
| [`publish-skill`](skills/publish-skill/) | Publish installed local agent skills into a GitHub skills catalog repo. |
| [`youtube-script-extractor`](skills/youtube-script-extractor/) | Extract scripts, captions, subtitles, or transcript text from YouTube video URLs. |
